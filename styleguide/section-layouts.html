3 #layouts Layouts

The layout styling for major parts of the page that are included with the theme. Note: some Panels layouts are included in other parts of the system and are not documented.

3.1 #layouts.sidebars Responsive sidebars layout

Positioning for a responsive layout.

Define CSS classes to create a fluid grid layout with optional sidebars depending on whether blocks are placed in the left or right sidebars.

This layout uses the Zen Grids plugin for Compass: http://zengrids.com
